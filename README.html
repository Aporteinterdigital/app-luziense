Aplicativo Municipal - Port Interdigital

Este repositório contém uma versão web do aplicativo municipal desenvolvido para a Port Interdigital. O aplicativo foi projetado para fornecer informações e serviços aos munícipes, incluindo avaliação de transporte, marketplace local, clube de benefícios e sistema de denúncias.

Esta é uma versão de demonstração web do aplicativo, criada para visualização do design e das funcionalidades. A versão completa para dispositivos móveis foi desenvolvida em Flutter.

Funcionalidades

  • Tela Inicial - Acesso rápido a todas as funcionalidades, notícias e eventos da cidade.
  • Avaliação de Transporte - Sistema para avaliar mototáxis e carros de aplicativo.
  • Marketplace Local - Plataforma para vendedores da região oferecerem seus produtos e serviços.
  • Clube de Benefícios - Sistema de pontos e vantagens para incentivar compras na região.
  • Denúncias e Reclamações - Canal para reportar problemas na cidade.

Tecnologias Utilizadas

  • HTML5
  • CSS3
  • JavaScript

Como Visualizar

Para visualizar o aplicativo, basta acessar: https://portinterdigital.github.io/app-municipal

Alternativamente, você pode clonar este repositório e abrir o arquivo index.html em seu navegador:

git clone https://github.com/portinterdigital/app-municipal.git
cd app-municipal
# Abra o arquivo index.html em seu navegador

Versão Completa

A versão completa do aplicativo foi desenvolvida em Flutter, permitindo sua execução em dispositivos iOS e Android. O código-fonte da versão Flutter está disponível em um repositório separado.

Cores e Design

O aplicativo segue uma paleta de cores específica:

  • Dourado (#D4AF37) - Utilizado para botões de ação e elementos de destaque
  • Azul (#1E88E5) - Utilizado para cabeçalhos e elementos principais
  • Preto (#000000) - Utilizado para textos e contrastes
